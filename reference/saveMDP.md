# Save the MDP to binary files

Currently do not save external files.

## Usage

``` r
saveMDP(mdp, prefix = "", getLog = TRUE)
```

## Arguments

- mdp:

  The MDP loaded using
  [`loadMDP()`](http://relund.github.io/mdp/reference/loadMDP.md).

- prefix:

  A character string with the prefix added to `binNames`. Used to
  identify a specific model.

- getLog:

  Output the log as a message.
